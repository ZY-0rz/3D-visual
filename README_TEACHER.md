# 老师验收使用说明（网页版）

本网页支持在浏览器本地读取 `f3_sm.sgy` 与 `new_dem.tif`，并在**不上传文件**的前提下将 SEGY 转为 VTI，再进行可视化展示。

## 使用步骤

1. 打开网页链接（老师只需要浏览器即可）。
2. 在“地形（DEM）”面板点击“加载 DEM”，选择 `new_dem.tif`。
3. 在“地震（SEGY）”面板点击“加载 SEGY/VTI”，选择 `f3_sm.sgy`。
4. 加载完成后会自动开始“SEGY→VTI”转换：
   - 面板会显示“转换中 xx%”
   - 转换完成后出现“下载 VTI”链接（可下载生成的 `.vti` 文件）
5. 可视化：
   - 3D 视图可切换“体渲染 / 正交切片 / 等值面”
   - 2D 视图可查看“变密度 / 变面积（wiggle）”剖面，并支持鼠标缩放与拖拽平移

## 重要说明

- 转换与渲染均在老师本机浏览器中完成，不会把数据上传到服务器。
- `f3_sm.sgy` 文件较大，首次加载/转换需要一定时间与内存，建议使用 Chrome/Edge，并保持电脑内存充足。
